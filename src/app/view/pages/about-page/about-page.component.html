About this Application

This application was built by Arjune Mirchandani using Angular 12, as a demonstration of the advantages of the Angular framework. and utilizes the free iTunes RSS feed to demonstrate data api retrieval and handling. This application has two main views:

  • Table View
  • Grid View

The location of the real-time RSS Feed that was used in this demonstration is the following:

  • iTunes top 100 Albums RSS

This application has the following features:

  • Utilizes the Angular Framework
  • A clean, modern look
  • Clean, maintainable codebase
  • Cross browser and cross platform support
  • Responsive Design
    • Utilizes the Angular Flex Layout module
  • Implements a good user experience

Additionally, this application demonstrates the following Angular Material component usage:

  • Mat Drawer
  • Mat Table, Paginator, and Sort
  • Mat Dialog
  • Mat Snackbar
  • Material Form Input components
  • ... and more!

Furthermore, it features the following:

  • Useful & meaningful comments where needed
  • CSS animations
  • SASS mixins integration
  • Album details functionality
  • Album favorites manager
  • Implements NGRX state management

Hosting Information:

  • Hosted publicly on my AWS EC2 C5.large server instance running CentOS 7 w/ Plesk
